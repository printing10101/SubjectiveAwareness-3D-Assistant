## [1.0.0] - 2026-06-15

### 新增

#### 前端重构 (V1.0.0)

- **设计系统**：建立完整的设计令牌体系（颜色、间距、圆角、阴影、动效），定义在 `tokens.css`
- **响应式布局**：实现三档响应式断点（移动端 <768px / 平板端 768-1279px / 桌面端 ≥1280px）
- **布局组件**：新增 `AppLayout`、`AppSidebar`、`AppHeader`、`MobileTabbar`、`PageContainer` 布局组件
- **UI 基础组件库**：新增 19 个 UI 基础组件（BaseButton、BaseInput、BaseCard、BaseModal、BaseDrawer、BaseToast 等）
- **动效系统**：建立统一的页面过渡、卡片悬停、按钮交互、模态框、抽屉、Toast 等动画规范
- **无障碍支持**：添加 WCAG 2.1 AA 级焦点环样式、skip-to-main 链接、高对比度模式支持
- **工具类系统**：新增间距、文字色、背景、圆角、阴影等工具类（`utilities.css`）
- **知识库模块**：新增知识库管理页面（列表、详情、编辑、知识图谱可视化）
- **案件管理模块**：新增案件列表、案件详情、相似案件检索页面
- **评估中心**：新增评估中心页面（admin 权限）
- **案件标注**：新增案件标注界面（admin/analyst 权限）
- **用户认证**：新增登录、权限守卫、协议接受验证
- **图片优化**：新增 `ResponsiveImage` 组件，支持懒加载和淡入效果

### 修改

- **路由重构**：统一使用 `AppLayout` 包裹主页面路由，移除独立布局页面
- **样式架构**：从分散的 CSS 文件迁移至统一的设计令牌系统（`tokens.css` + `base.css` + `utilities.css` + `animations.css`）
- **全局样式**：优化 Reset 样式，统一滚动条、选区、焦点环等全局视觉风格
- **状态管理**：重构 Pinia store，拆分为 `auth`、`case`、`analysisStore`、`knowledgeStore` 四个模块
- **构建优化**：配置 Vite 代码分割策略，按功能模块拆分 vendor chunk
- **压缩配置**：启用 Gzip + Brotli 双算法压缩（阈值 10KB）

### 删除

- **临时文件**：删除 `_LayoutTestView.vue`（布局测试页面）
- **临时路由**：移除 `/__layout-test` 测试路由
- **冗余样式**：清理 `global.css` 中与 `base.css` 重复的 Reset 规则
- **旧变量文件**：`variables.css` 标记为过渡期兼容，新代码统一使用 `tokens.css` 变量

### 修复

- 修复移动端侧边栏折叠状态下的布局偏移问题
- 修复骨架屏动画在 `prefers-reduced-motion` 下未正确禁用的问题
- 修复焦点环在深色背景下对比度不足的问题（调整 outline-color）

### 性能优化

- CSS 代码分割：启用 Vite `cssCodeSplit`，按需加载组件样式
- 资源内联：4KB 以下资源自动内联，减少 HTTP 请求
- 字体优化：使用 `font-display: swap` 避免 FOIT
- 图片优化：支持 `loading="lazy"` 懒加载 + 淡入效果

## 格式说明

每个版本按以下分类记录变更：

- **新增**：新功能、新组件、新页面
- **修改**：对现有功能的改进或调整
- **删除**：移除的功能、组件或文件
- **修复**：Bug 修复
- **性能优化**：性能相关的改进
- **兼容性**：可能影响现有功能的变更

每条记录包含：变更类型、影响范围（组件/模块名）、简要说明。
