# 案例三：老人被骗案（确实不明知情形）

## 一、被害人基本信息

| 项目 | 内容 |
|------|------|
| **姓名** | 王某 |
| **年龄** | 68岁 |
| **身份** | 退休职工（原XX市纺织厂工人） |
| **文化程度** | 高中毕业 |
| **无前科记录** | 是 |
| **健康状况** | 轻度高血压，无认知障碍 |
| **家庭状况** | 独居，子女在外地工作 |

---

## 二、案件事实概述

被害人王某于**2023年10月**期间，接到**冒充公检法机关**的诈骗电话，对方以"涉嫌洗钱需资金核查"为由进行欺骗，王某在惊恐状态下将银行卡密码告知对方，并按要求办理了多张银行卡。涉案银行卡被诈骗分子用于接收诈骗资金**2000余万元**，王某本人**未从中获利**，发现异常后**主动报案**。

---

## 三、详细事实经过

### 3.1 诈骗电话内容

| 时间 | 事件 | 详情 |
|------|------|------|
| 2023年10月12日上午9:30 | 接到诈骗电话 | 来电显示为"XX市公安局"（号码经伪基站伪造） |

#### 诈骗话术还原

> **诈骗分子**："您好，这里是XX市公安局刑侦支队，我姓刘，警号XXXXXX。我们接到中国人民银行反洗钱中心通报，您名下的一张银行卡涉嫌一起重大洗钱案件，涉案金额高达800余万元。"

> **王某**："不可能！我怎么可能洗钱？"

> **诈骗分子**："我们理解您的心情，但这是上级机关转下来的案件。您的身份信息可能被盗用，现在需要您配合调查。为了不影响您的正常生活，这个案件目前暂时保密，请您不要告诉任何人，包括您的家人。"

> **诈骗分子**："现在需要您配合进行资金核查，请您携带身份证到以下银行办理新的银行卡：中国工商银行、中国建设银行、中国农业银行。办理完成后，将卡号和密码告知我们，我们将对您的资金进行核查。如果查清没有问题，您的嫌疑就会解除。"

> **诈骗分子**："如果您不配合，我们将依法对您采取强制措施，包括冻结您所有账户、限制出境，甚至可能对您进行刑事拘留。"

#### 密码泄露过程

王某在办理完银行卡后，按照诈骗分子要求：

1. **电话告知密码**：通过手机电话逐一告知6张银行卡的密码
2. **开通网银**：按对方要求开通手机银行和网上银行
3. **提供验证码**：收到银行短信验证码后，按对方要求念出验证码
4. **屏幕共享**：在对方指导下下载并开启屏幕共享软件，对方可直接操作其手机银行

## 四、主观认知状态分析

### 4.1 年龄与认知能力因素

| 因素 | 具体描述 |
|------|----------|
| **年龄因素** | 68岁，属于老年人群体，对新型诈骗手段识别能力较弱 |
| **认知能力** | 无认知障碍，但对电信诈骗缺乏了解 |
| **信息获取渠道** | 主要通过电视、报纸获取信息，较少接触网络防诈骗宣传 |
| **心理状态** | 独居老人，对"公检法"机关有天然的信任感和敬畏感 |

### 4.2 对"公检法"身份的信任程度

王某在多次询问中始终表示：

1. **来电显示为"XX市公安局"**，增强了其信任度
2. **对方能准确说出其个人信息**（身份证号、住址、银行账号等）
3. **对方使用了专业的法律术语**（"反洗钱"、"资金核查"、"强制措施"等）
4. **对方声称可以出示"警官证"和"通缉令"**（通过短信发送伪造图片）
5. **对方威胁不配合将采取强制措施**，使其产生恐惧心理

### 4.3 一贯陈述

王某从报案至今，在所有询问笔录中**始终坚称**：

> **"我相信对方是警察。"**

其核心观点包括：

- 从未想过对方是骗子
- 配合办卡是因为相信自己在"配合调查"
- 告知密码是因为对方说"需要核查资金"
- 从未想过自己的银行卡会被用于犯罪
- 对涉案2000余万元资金完全不知情

## 五、法律分析

### 5.1 主观认知分析

| 分析维度 | 结论 |
|----------|------|
| **是否具有主观明知** | **否**。王某系被骗取银行卡信息，主观上认为自己在配合公安机关调查 |
| **是否具有犯罪故意** | **否**。王某没有参与犯罪的故意，其行为系被诈骗分子利用 |
| **是否构成犯罪** | **不构成**。王某系诈骗案件的被害人，而非犯罪嫌疑人 |

### 5.2 法律依据

根据《中华人民共和国刑法》及相关司法解释：

1. **主观方面**：构成帮助信息网络犯罪活动罪需要行为人**主观上明知**他人利用信息网络实施犯罪。本案中，王某系被诈骗分子欺骗，主观上不具有明知。

2. **客观方面**：虽然王某的银行卡被用于转移诈骗资金，但其本人**未从中获利**，且系在被骗状态下提供银行卡信息。

3. **被害人身份**：王某本身系电信诈骗的被害人，其合法权益受到侵害，依法应当得到保护。

### 5.3 与"帮信罪"的界限

| 对比项 | 帮信罪行为人 | 本案王某 |
|--------|--------------|----------|
| 主观认知 | 明知他人实施犯罪 | 相信自己在配合公安调查 |
| 获利情况 | 按比例提成或收取费用 | 分文未得 |
| 行为主动性 | 主动提供银行卡 | 被骗提供银行卡信息 |
| 身份认定 | 犯罪嫌疑人/被告人 | 诈骗案件被害人 |

### 5.4 结论

**王某不构成任何犯罪**，其系电信诈骗案件的被害人。公安机关已对其报案的诈骗案件立案侦查，并追查涉案资金流向。

## 七、案件启示

1. **老年人群体是电信诈骗的高危人群**，应加强针对性的防诈骗宣传教育
2. **冒充公检法机关**是常见的诈骗手段，公安机关办案不会通过电话要求提供银行卡密码
3. **任何要求保密、威胁恐吓**的电话都应引起高度警惕
4. **发现异常后及时报案**是挽回损失的关键，王某在发现异常后第一时间报案，为公安机关追查资金争取了时间
