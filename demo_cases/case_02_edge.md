# 案例二：兼职刷单案（边缘认知情形）

## 一、被告人基本信息

| 项目 | 内容 |
|------|------|
| **姓名** | 李某 |
| **年龄** | 35岁 |
| **身份** | 全职主妇 |
| **文化程度** | 初中毕业 |
| **无前科记录** | 是 |
| **家庭状况** | 丈夫在外地打工，独自抚养两名未成年子女 |

---

## 二、犯罪事实概述

被告人李某于**2023年6月至2023年8月**期间，通过手机短视频平台接触到"网络兼职刷单"广告，在他人远程指导下，提供本人名下**2张银行卡**用于所谓"商家资金流转"，累计接收并转账资金**80余万元**，从中获利**2000元**。

---

## 三、详细事实经过

### 3.1 接触渠道

| 时间 | 事件 | 平台 |
|------|------|------|
| 2023年5月28日 | 在刷短视频时看到"在家兼职刷单，日赚百元"广告 | 某短视频平台 |
| 2023年5月29日 | 添加广告中留下的微信客服号 | 微信 |
| 2023年5月30日 | 被拉入"电商刷单兼职群"，群内成员约200人 | 微信群 |

### 3.2 广告内容描述

广告页面显示：

> **"正规电商刷单兼职，足不出户，日赚100-300元！**
> 
> - 工作内容：帮助商家完成订单刷量，提升店铺信誉
> - 要求：提供本人银行卡用于商家资金流转
> - 收益：每完成一笔刷单可获得50-200元佣金
> - 保证：合法合规，商家对公账户操作"

### 3.3 银行卡使用过程

#### 3.3.1 提供银行卡

| 时间 | 银行 | 卡号后四位 | 用途说明 |
|------|------|------------|----------|
| 2023年6月5日 | 中国农业银行 | 8821 | 被告知用于"接收商家刷单资金" |
| 2023年6月5日 | 中国邮政储蓄银行 | 3356 | 被告知用于"转账确认订单" |

#### 3.3.2 操作方式

1. **远程指导**：对方通过微信语音通话，一步步指导李某操作手机银行
2. **操作流程**：
   - 第一步：账户收到一笔款项（对方称为"商家刷单资金"）
   - 第二步：李某按照微信指令，将款项转至指定账户
   - 第三步：转账完成后截图发送给对方
   - 第四步：对方通过微信红包或小额转账支付"佣金"（通常50-100元/笔）
3. **操作频率**：平均每天操作3-5笔，持续约2个月

## 四、主观认知状态分析

### 4.1 个人背景因素

| 因素 | 具体描述 |
|------|----------|
| **文化程度** | 初中毕业，缺乏法律知识和金融常识 |
| **社会经验** | 长期在家照顾孩子，社会接触面窄，无类似犯罪经历 |
| **经济状况** | 家庭经济压力较大，希望通过兼职增加收入 |
| **认知能力** | 对"刷单"、"资金流转"等概念缺乏专业理解 |

### 4.2 主观认知状态

#### 李某的辩解

> **"我一直以为是在做正常的电商刷单。"**

李某在接受公安机关讯问时始终坚称：

1. 自己是通过正规渠道（短视频平台广告）接触到的"兼职"
2. 对方明确告知是"帮助商家完成订单刷量"
3. 群内有200多人都在做同样的事，感觉是正规兼职
4. 自己从未听说过"跑分"、"洗钱"等概念
5. 对方说是"合法合规"的，自己也问过是否违法，对方给予了肯定答复

### 4.3 主观明知的认定难点

| 认定难点 | 具体分析 |
|----------|----------|
| **1. 欺骗性话术** | 对方以"电商刷单"为名，使用了看似正当的商业话术，普通人难以辨别真伪 |
| **2. 群聊规模效应** | 200人的微信群营造出"大家都在做"的假象，降低了李某的警惕性 |
| **3. 知识水平限制** | 初中文化程度使其难以理解资金异常流转背后的法律风险 |
| **4. 无主动规避行为** | 李某未使用加密通讯工具，未采取任何隐蔽措施，均通过日常社交软件操作 |
| **5. 获利金额较低** | 2个月仅获利2000元，与专业"跑分"人员的获利水平相差悬殊 |
| **6. 无前科劣迹** | 李某此前无任何违法犯罪记录，系初犯 |

## 五、法律争议焦点

### 5.1 主观明知的证明标准

根据刑法规定，构成**帮助信息网络犯罪活动罪**需要行为人**主观上明知**他人利用信息网络实施犯罪。本案的争议焦点在于：

#### 控方观点

1. 李某作为成年人，应当知道提供银行卡给他人使用存在风险
2. 资金快进快出的模式明显异常，正常人应当产生怀疑
3. 对方要求转账至不同账户，且账户频繁更换，应引起警觉
4. 2000元获利虽然不高，但与其劳动付出明显不成比例

#### 辩方观点

1. 李某被"电商刷单"的话术所欺骗，主观上认为是正当兼职
2. 其文化程度和社会经验限制了其辨别能力
3. 无证据表明李某曾搜索或了解过"跑分"、"洗钱"等概念
4. 未采取任何隐蔽措施，说明其没有犯罪故意
5. 获利金额极低，不符合职业"跑分"人员的特征

### 5.2 法律适用分析

根据《最高人民法院、最高人民检察院关于办理非法利用信息网络、帮助信息网络犯罪活动等刑事案件适用法律若干问题的解释》：

- 本案**不完全符合**第十一条列举的"明显明知"情形
- 但需要综合考量李某的**认知能力、行为方式、获利情况**等因素
- 司法实践中，此类案件存在**罪与非罪的较大争议**

### 5.3 可能的处理结果

| 情形 | 处理结果 |
|------|----------|
| 认定主观明知成立 | 构成帮助信息网络犯罪活动罪，但可从轻处罚 |
| 认定主观明知证据不足 | 不构成犯罪，但可能面临行政处罚 |
| 认定情节显著轻微 | 依法免予刑事处罚 |
