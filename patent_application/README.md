## 一、材料包说明

本目录包含「**缘法而智——帮信罪主观明知动态建模与辅助分析系统**」发明专利申请所需的**全部材料**，可直接提交给**专利代理机构**用于撰写与提交申请，也可由发明人本人通过**国家知识产权局专利业务办理系统**（[https://cponline.cnipa.gov.cn](https://cponline.cnipa.gov.cn)）电子提交。

### 1.1 涉及的法律依据

- 《中华人民共和国专利法》（2020 年修正）
- 《中华人民共和国专利法实施细则》（2010 年修订）
- 《专利审查指南》（2010 年颁布，含 2019、2023 年更新）
- 《关于规范专利申请行为的若干规定》

### 1.2 涉及的业务领域

- 发明类型：发明专利（针对系统+方法）
- 技术领域：人工智能（G06N）、司法智能化（G06Q50/18）
- 司法业务：刑事司法 / 帮信罪 / 主观明知认定

## 四、提交方式

### 4.1 电子申请（推荐）

1. 登录国家知识产权局**专利业务办理系统**：[https://cponline.cnipa.gov.cn](https://cponline.cnipa.gov.cn)；
2. 注册账号、完成实名认证；
3. 进入"专利申请"模块；
4. 依次填写：请求书 → 说明书 → 权利要求书 → 摘要 → 附图；
5. 上传相关文件（PDF 格式）；
6. 缴纳申请费；
7. 提交申请，获取申请号与申请日。

### 4.2 纸质申请

1. 将所有材料按规范打印为 **A4 纸**；
2. 说明书附图单独打印，每张占一页 A4；
3. 前往国家知识产权局**专利局受理大厅**或**地方代办处**窗口提交；
4. 缴纳申请费；
5. 取得受理通知书。

### 4.3 委托代理机构

1. 与专利代理机构签订《专利代理委托书》；
2. 提交本材料包 + 项目源代码 + 现有技术检索报告（如有）；
3. 由专利代理师完成撰写、校对、提交；
4. 配合答复审查意见通知书；
5. 授权后缴纳登记费与年费。

## 六、申请流程时间表（参考）

| 阶段 | 时间节点 | 重要事项 |
|------|----------|----------|
| T0 | 申请日 | 提交申请，获取申请号 |
| T0+18 个月 | 公布日 | 申请文件公开（默认；可请求提前公布） |
| T0+36 个月内 | 实质审查请求 | 提交实质审查请求并缴费 |
| T0+18~24 个月 | 审查意见 | 收到第一次审查意见通知书 |
| T0+24~30 个月 | 授权或驳回 | 收到授权通知书或驳回决定 |
| T0+24~30 个月（驳回时） | 复审 | 3 个月内可提复审请求 |
| 授权后 | 登记与缴费 | 缴纳登记费与第 1-3 年年费 |
| 授权后 | 维持年费 | 第 4 年起按年缴纳年费维持专利权有效 |

## 七、注意事项

### 7.1 申请前检查

- [ ] 全部材料已完整填写
- [ ] 发明人/申请人信息真实、完整
- [ ] 附图清晰，文件命名规范
- [ ] 权利要求书与说明书术语一致
- [ ] 摘要不超过 300 字
- [ ] 已检索现有技术，确认本发明具有新颖性、创造性、实用性
- [ ] 已确认无保密问题（不涉及国防利益）

### 7.2 提交后检查

- [ ] 已缴纳申请费
- [ ] 已获取申请号与申请日
- [ ] 已在规定期限内（3 年）提交实质审查请求
- [ ] 收到审查意见后，已及时答复
- [ ] 授权后 2 个月内办理登记手续
- [ ] 已按时缴纳年费

### 7.3 常见驳回原因及预防

| 驳回原因 | 预防措施 |
|----------|----------|
| 不具备新颖性 | 提交前做好现有技术检索 |
| 不具备创造性 | 突出本发明与现有技术的实质区别 |
| 说明书公开不充分 | 实施例要充分、具体 |
| 权利要求不清楚 | 权利要求用语应与说明书一致、清晰 |
| 涉及国家安全 | 提前申请保密审查 |
