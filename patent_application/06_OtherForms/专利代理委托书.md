## 一、委托事项

委托人【______】（以下简称"委托人"）就**缘法而智——帮信罪主观明知动态建模与辅助分析系统**发明专利申请及相关事宜，委托受托机构【______】专利代理机构（以下简称"受托机构"）办理下列事项：

### （一）专利申请阶段

1. 撰写发明专利请求书、说明书、权利要求书、说明书摘要及说明书附图；
2. 向国家知识产权局提交发明专利申请并取得申请号、申请日；
3. 缴纳专利申请费、公布印刷费等各项费用；
4. 办理其他与专利申请相关的事宜。

### （二）审查阶段

1. 提交实质审查请求书并缴纳实质审查费；
2. 答复国家知识产权局发出的审查意见通知书；
3. 在审查过程中对申请文件进行修改（如需）；
4. 处理驳回、复审等事宜。

### （三）授权及后续阶段

1. 办理专利登记手续、缴纳专利年费；
2. 处理专利权无效宣告请求（如涉及）；
3. 办理专利权质押、转让、许可等事宜（需另行书面授权）；
4. 办理其他与专利权相关的事宜。

## 四、委托期限

本委托书自双方签字/盖章之日起生效，至下列任一情形出现时终止（以先发生者为准）：

1. 专利申请被驳回且未提起复审；
2. 专利权终止；
3. 委托人书面通知受托机构解除本委托；
4. 双方另行协商一致终止本委托。

## 五、双方权利与义务

### （一）委托人权利与义务

1. 有权了解专利申请的进展情况和审查结果；
2. 应如实向受托机构提供与发明相关的全部技术资料；
3. 应按照约定向受托机构支付代理费用；
4. 涉及发明人署名、权利要求修改等关键事项的，应书面确认；
5. 应配合受托机构完成各项官方手续。

### （二）受托机构权利与义务

1. 应按照《专利代理条例》和《专利代理管理办法》的规定，恪尽职守、依法代理；
2. 应保守委托人的技术秘密和商业秘密；
3. 应及时向委托人通报专利申请进展情况和审查意见；
4. 应妥善保管与本专利申请相关的全部文件资料；
5. 因受托机构过错造成委托人损失的，应依法承担相应责任。

## 七、保密条款

受托机构及专利代理师对本委托书项下涉及的所有技术资料、商业秘密负有保密义务，未经委托人书面同意，不得向任何第三方披露。

## 八、争议解决

因本委托书引起的或与本委托书有关的任何争议，双方应协商解决；协商不成的，提交**【______】仲裁委员会**仲裁或向**【______】人民法院**提起诉讼。

## 九、其他约定

1. 本委托书一式两份，双方各执一份，具有同等法律效力；
2. 本委托书未尽事宜，双方另行协商签订补充协议；
3. 本委托书自双方签字/盖章之日起生效。
