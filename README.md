# SubjectiveAwareness-3D-Assistant
帮信罪主观明知AI辅助分析系统——基于三维度结构化综合认定框架的司法辅助原型。 聚焦帮助信息网络犯罪活动罪中主观明知认定难、说理简化的痛点，将司法解释中的综合认定要素转化为“客观行为异常度—认知匹配度—辩解合理性”三个递进维度。采用本地部署的DeepSeek-R1大模型与规则引擎混合架构，实现案件事实抽取、维度归类、档级判断、冲突校验、类案参照及结构化报告生成。 系统定位为司法辅助工具，不替代司法人员独立判断，不改变证明责任分配，所有分析均锚定具体证据并支持人工复核。项目通过公开案例回溯实验验证可行性，旨在提升主观明知审查的说理规范性与透明度。
